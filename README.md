A production-ready, high-performance **Natural Language Processing (NLP)** application designed to ingest large-scale customer review datasets and deliver highly accurate binary sentiment classification.By leveraging a fine-tuned DistilBERT transformer model, this system captures deep contextual text semantics, heavily outperforming traditional machine learning baselines while maintaining ultra-low inference latency.


## Key Features & Architecture

**Transformer Fine-Tuning:** Optimized a pre-trained **DistilBERT** architecture on a comprehensive dataset of 25,000+ customer reviews for robust text classification**Baseline Benchmarking:** Elevated performance metrics by **14%** over standard baseline paradigms like Logistic Regression to validate deep learning efficacy**Rigorous Evaluation:** Secured a highly stable **90.4% validation accuracy** threshold coupled with a balanced **0.88 macro-F1** score **Interactive UI Deployment:** Built and hosted a real-time predictive web dashboard utilizing **Streamlit Cloud** for instantaneous model inference29]. **Token Optimization:** Managed highly optimized, dynamic sequence padding and tokenization pipelines to maximize computational efficiency during model inference


## Tech Stack

**Core Framework:** Hugging Face Transformers 
**Deep Learning Engines:** PyTorch / TensorFlow 
**Frontend Dashboard:** Streamlit 
**Data & Machine Learning:** Scikit-learn, Pandas, NumPy
**Language:** Python
